Output

Het plaatsen van tekst op het scherm

Een van de manieren om tekst op het scherm te zetten is een alert

Dit is nog een knop die een alert weergeeft

Je kunt ook tekst op het scherm zetten met document.write("Demo Tekst")

Wanneer je document.write() aanroept nadat de pagina geladen is wordt alles overschreven

Je kunt naar de console schrijven met console.log()

We gaan nu tekst veranderen van een p-tag op de pagina

Deze tekst wordt verandert nadat er op de knop hiernaast is gedrukt

Maak een input-tag waar je tekst in kunt plaatsen. Maak een knop. Maak een paragraaf-tag. Wanneer je op de knop drukt wordt de ingevoerde tekst weergegeven in de paragraaf-tag.

Comments

De eerste manier om in javascript code uit te commentarieren is per regel. Je zet dan voor de coderegel die je wilt uitcommentarieren twee forwardslashes // De browser negeert dan deze code. In Visual studio code wordt je code groen.

De tweede manier om in javascript code uit te commentarieren is per codeblock. Je zet voor het codeblock een een forwardslash en een asterisk /* en aan het einde van het codeblock een asterisk en een forwardslash */. De browser negeert dan deze code. In Visual studio code wordt je code groen.

Let op! HTML commentarieer je uit op een andere manier:
<!‐‐ niet zichtbaar ‐‐>
